# BingTraslatorX

A Chrome extension for Chiese-English/English-Chinese translating. Support mouse hovering and selection.

基于必应词典的中英互译, 支持鼠标悬浮取词、英美音标和发音。如果你喜欢必应词典，不要错过哦! 😊
